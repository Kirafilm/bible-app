### 技術架構
- React Native + Expo SDK 52（不用 Expo Router，改用簡易 Tab 切換）

- AsyncStorage（本地儲存書籤和閱讀進度）
- 字體大小調整、書籤、搜尋、經文分享

### 依賴安裝問題與解決
- 最初嘗試用 Expo Router v4 + file-based routing，但 `expo-router` 依賴的 `react-native-screens@4.25.2` 要求 `react-native >= 0.82`，與 Expo SDK 52（`react-native 0.76`）衝突
- 解決方案：改用簡易版 `App.tsx` + `screens/` 目錄，不用 `expo-router`，依賴問題完全解決
- `package.json` 最終只包含：`expo`, `react`, `react-native`, `async-storage`, `safe-area-context`
